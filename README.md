# Java Inventory Management System

A console-based Inventory Management System developed using Java and Object-Oriented Programming principles.

## Features

- Add Product
- View Products
- Update Product
- Delete Product
- Exception Handling
- ArrayList-based Data Storage

## Technologies Used

- Java
- OOP
- Collections Framework
- Exception Handling

## Project Structure

- Product.java
- InventoryManager.java
- Main.java

## Concepts Demonstrated

- Classes and Objects
- Encapsulation
- ArrayList
- CRUD Operations
- Exception Handling
